# Guia de Introdução ao GitHub para Novos Desenvolvedores

Este guia foi criado para orientar os novos integrantes da equipe no uso do Git e GitHub. Siga estes passos para configurar seu ambiente, baixar o projeto e enviar suas contribuições.

> [!IMPORTANTE]
> Este projeto utiliza o **GitHub**. Você precisará de uma conta no GitHub e permissão de acesso ao repositório.

## Passo 1: Preparando o Ambiente (Apenas na primeira vez)

1.  Crie uma pasta em seu computador onde você quer guardar o projeto.
2.  Abra o Google Antigravity e indique que vai trabalhar na pasta criada para o projeto.
3.  Você precisa ter o **Git** instalado no seu computador, escreva no prompt do Antigravity: "Baixe e instale o Git no meu pc local".


## Passo 2: Baixando o Projeto (Clonar)

Para baixar os arquivos do projeto para o seu computador (chamamos isso de "clonar"):
Escreva no prompt do Antigravity: clone https://github.com/USUARIO-DO-PROJETO/NOME-DO-REPOSITORIO.git

> [!TIP]
> Substitua `USUARIO-DO-PROJETO` e `NOME-DO-REPOSITORIO` pelos valores corretos do projeto que você vai colaborar.

## Passo 3: Entrando na Sua Branch

Cada desenvolvedor tem uma "branch" (ramificação) própria para trabalhar sem atrapalhar os outros. **Nunca trabalhe direto na branch `main`.**

Para mudar para a sua branch (substitua `sua-branch` pelo nome que criaram para você no GitHub):

Escreva no prompt do Antigravity: git checkout sua-branch

DICAS:
> Se não lembra o nome exato da sua branch, peça pro Antigravity: liste os nomes das branchs deste projeto no GitHub.
> 
> Para confirmar em qual branch você está, digite `git status`. Deve aparecer "On branch sua-branch".
> 
> Avise o Gravity pra não esquecer: a partir de agora, toda vez que eu abrir este projeto, todas as alterações que eu fizer devem ser feitas na branch com nome **sua-branch**

## Passo 4: Trabalhando e Salvando Alterações

Agora você pode criar novos arquivos ou editar os existentes.

Quando terminar uma tarefa e quiser salvar no histórico do Git, escreva no prompt do Antigravity:
Faça commit e push de todos os arquivos modificados, adicionando uma mensagem informando as principais alterações realizadas.

* Se for a primeira vez, pode pedir seu usuário e senha do GitHub.

## Passo 5: Rodando o sistema no seu PC para testar as funcionalidades

Escreva no prompt do Antigravity: quando eu pedir aqui para rodar o sistema, utilize o seu terminal para iniciar o servidor e me avise quando posso acessar o endereço.
Assim, nos próximos testes, só precisará escrever : rodar o sistema.
